.. py:class:: Meta(popmetabinpath=None, datapath=None, metaXSLTDirectory=None, dump_meta=False, TSV_output=True, prefixTSV=None, PHYLIP_output=False, ihwg_output=False, batchsize=0, outputDir=None, xml_files=None)

   Aggregates output from multiple population runs.

   Transform a specified list of ``.xml`` output files to ``.tsv``
   tab-separated values (TSV) form.

   :param popmetabinpath: the directory for where meta sources are kept
   :type popmetabinpath: str
   :param datapath: data where XSLT and other meta sources may be kept
   :type datapath: str
   :param metaXSLTDirectory: fallback XSLT directory
   :type metaXSLTDirectory: str
   :param dump_meta: create the ``meta.xml`` file (default to ``False,``)
   :type dump_meta: bool
   :param TSV_output: output ``.tsv`` tables by default (enabled by default). (such tables can be used by R)
   :type TSV_output: bool
   :param prefixTSV: prefix to use for all ``.tsv`` files
   :type prefixTSV: str
   :param PHYLIP_output: create PHYLIP output (disabled by default)
   :type PHYLIP_output: bool
   :param ihwg_output: by default, don't enable the 13th IHWG format headers
   :type ihwg_output: bool
   :param batchsize: size of batches to process separately (default
                     ``batchsize=0``, a separate batch for each file)
   :type batchsize: int
   :param outputDir: output directory to write XML files to
   :type outputDir: str
   :param xml_files: list of generate XML files
   :type xml_files: list
