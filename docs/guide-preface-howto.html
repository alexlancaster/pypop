2. How to use this guide

This guide to PyPop contains three parts:

  • Chapter 1, Installing PyPop describes how to download and install a standalone binary version of the application. Programmers and other interested parties can download source and can build the application themselves.

  • Chapter 2, Getting started with PyPop describes how to run PyPop.

  • Chapter 3, Interpreting PyPop output details the population genetic methods and statistics that PyPop computes.
